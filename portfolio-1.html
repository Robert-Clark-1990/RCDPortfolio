Milly Armstrong Clark

Robert Clark - November 2020 Awarded Merit grade by The Code Institute

My first milestone project in my Diploma in Software Development, and first foray into coding was the creation of a website for comedian/podcaster/screenwriter, Milly Armstrong Clark. The design features a fun and colourful aesthetic in keeping with Milly's comedic style, and showcases a selection of her work, a news page, and a contact form.

This project was built using the five planes of UX design to develop a site that is focused in its needs and objectives, and provides a high quality experience for the user. Information on features, technologies used, testing and deployment can be found on the project's GitHub page.
