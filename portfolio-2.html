Viking Adventure

Robert Clark - December 2020 Awarded Distinction grade by The Code Institute

The second project I worked on, and inspired by classic point and click adventure games such as the Monkey Island series, Viking Adventure is a short demo showcasing the powers of Javascript and JSON.Players guide main character Ivar as he attempts to win the favour of three high council members of a Norse settlement.

I created this project in order to test my skills in Javascript. Information on features, JSON file implementation, technologies used, testing and deployment can be found on the project's GitHub page.
