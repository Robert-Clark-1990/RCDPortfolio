AT Engineering Solutions

Robert Clark - July 2021

Technologies Used:

  • HTML5
  • CSS3
  • Javascript
  • JQuery
  • Python
  • Django
  • Allauth
  • AWS

Marking my first efforts in web development for a client (albiet a cousin), AT Engineering Solutions is a website designed to showcase the engineering and metalwork of the team, and encourage users to contact them if they have work.

This site uses Elfsight to automatically populate an up-to-date feed of the business's Instagram account (per the client's request). However, every other element on the page has been created by me. The project was created with Python, and has a dedicated backend admin page for the client to upload new customer reviews when necessary.

All design elements, including the AT Engineering Solutions logo, was created by me. Information on features, technologies used, testing and deployment can be found on the project's GitHub page.
