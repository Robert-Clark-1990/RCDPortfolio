AT Engineering Solutions

Robert Clark - July 2021

Marking my first efforts in web development for a client (albiet a cousin), AT Engineering Solutions is a functioning website designed to generate work from aspiring clients.

This site uses Elfsight to automatically populate an up-to-date feed of the business's Instagram account (per the client's request). However, every other element on the page has been created by me. The project runs with Python, and has a dedicated backend admin page for the client to upload new customer reviews when necessary.

Information on features, technologies used, testing and deployment can be found on the project's GitHub page.
